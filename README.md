# hd-lead-conversion-modelling
Python notebooks for MSc consultancy project

## HD Lead Conversion Modelling – Python Notebooks

This repository contains the Python notebooks used for an MSc Business Analytics consultancy project.

### Notebooks
- **01_core_modelling.ipynb**  
  Lead-level conversion modelling and agent–lead allocation analysis.

- **02_eda_descriptive.ipynb**  
  Descriptive exploratory analysis of leads and campaigns.

- **03_diagnostics_matching.ipynb**  
  Diagnostics examining unmatched sales and data linkage limitations.

### Outputs
All figures and tables generated by the notebooks are saved in the corresponding output folders.
